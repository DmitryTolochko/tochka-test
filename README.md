# Ссылка на готовый собраный проект: https://tochka-test.web.app/

## Как запустить локально?
npm install dependencies
npm run dev

## Реализованные фичи
1) Бот-соперник на алгоритме minimax
2) Настройки: размер поля, количество фишек для победы, отключение бота
3) Хранение результатов (в localStorage чтобы рефреш страницы сохранял состояние игры)
4) Анимации падения фишек, отображения выигрышной комбинации
5) Верстка адаптируется под экран

+ Вертска по БЭМ
+ JSDoc документация функций
+ Роутинг через react-router-dom
+ Хранилище без использования сторонних библиотек
+ Деплой с помощью firebase
